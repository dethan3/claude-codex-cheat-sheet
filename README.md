# Claude Code 中文速查表

这是一个对 [cc.storyfox.cz](https://cc.storyfox.cz/) 的中文本地化复刻版本，保留原站的单页排版、打印友好布局和少量交互，并将主要文案翻译为更符合中文开发者阅读习惯的表达。

当前页面基于原站显示的版本信息整理：

- Claude Code `v2.1.81`
- 最近更新时间：`2026-03-24`

## 特点

- 单页静态站点，无构建步骤
- 保留原站四栏速查布局
- 保留 A4 横向打印样式
- 保留 Mac / Windows 键位切换
- 保留 `NEW` 徽标自动过期逻辑
- 去除了外部统计脚本

## 文件结构

- `index.html`：页面结构与全部内容
- `styles.css`：页面样式和响应式/打印样式
- `script.js`：键位切换与 `NEW` 徽标逻辑

## 本地使用

最简单的方式是直接用浏览器打开 `index.html`。

如果你希望通过本地服务访问，也可以在仓库目录运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/`。

## 翻译原则

- 命令、参数、路径、环境变量、Slash Commands、Skills、Agents 等专有内容尽量保留原文
- 功能描述改写为中文开发者更常用的表达
- 避免逐字硬译，优先保证查阅效率

## 说明

- 这不是官方版本，只是面向中文读者的复刻与翻译
- 页面内容以原站抓取时的状态为准，后续如果原站更新，中文版本需要手动同步
